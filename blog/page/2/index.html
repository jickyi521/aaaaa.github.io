在mac mini 和 mbp 上安装过程额外发现的问题如下:

bundle install 安装失败

先安ruby的rvm环境

rvm --force install 1.9.2  
gem install bundle --no-ri --no-rdoc
bundle install

Reference

Zsh下octopress命令不识别

rake new_post["test"]
->zsh: no matches found: new_post[test]

*1、 由于zsh的通配符造成的 *, (, |, <, [, ?, 可以用引号括起来 
rake "new_post[test]"

*2、要根治的话需要改zsh配置文件 加 alias rake="noglob rake" 取消通配
